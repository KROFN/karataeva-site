# Karataeva Site

Finished photographer landing page built as a client-facing frontend/design project.

> **Status:** Finished portfolio project.  
> Created as a practical frontend/design case for a real personal brand scenario.

## What it is

Karataeva Site is a visual landing page for a photographer. The project focuses on atmosphere, clean presentation, responsive layout and a simple client-facing structure.

## What it shows

- landing page structure for a service-based personal brand
- visual-first frontend design
- responsive React/Vite implementation
- animation and interaction polish
- practical portfolio-oriented delivery

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- GSAP

## Run locally

```bash
npm install
npm run dev
```

## Notes

This repository is part of my public web portfolio. The project is kept as a finished frontend/design case rather than an actively maintained product.
